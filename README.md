<h4>
	<span style="font-size:16px;font-family:'Microsoft YaHei';">为什么叫QuickBack？</span> 
</h4>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';">因为我自己经常有整理文档的习惯，不仅需要写文档方面，而且能够快速回顾知识点，所以格式上word和txt文件都远不html方便，管理上文件系统远不如一个网站；之所以命名为QuickBack，意思是快速的回顾到知识点，毕竟人的记忆是有限的。</span> 
</p>
<p>
	<br />
</p>
<h4>
	<span style="font-size:16px;font-family:'Microsoft YaHei';">为什么不用wordpress？</span> 
</h4>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';">因为wordpress太重了，QuickBack的代码不到2000行，还不到wordpress的一个文件的代码量；而且QuickBack采用MVC的设计模式，方便开发者在QuickBack的基础上继续开发；目前代码已经上传到github，大家都可以建立新的分支；</span> 
</p>
<p>
	<br />
</p>
<h4>
	<span style="font-size:16px;font-family:'Microsoft YaHei';">QuickBack的特点：</span> 
</h4>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';">1. 数据库只有两张表，一张用来记录文章，另一个表用来记录网站日志；</span> 
</p>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';">2. 可以关注下左边菜单，有几处人性化的设计；</span> 
</p>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';">3. 标签用英文逗号分开，标签是自动生成的；</span> 
</p>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';"><br />
</span> 
</p>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';"></span>

	<span style="font-size:16px;font-family:'Microsoft YaHei';">安装说明：</span> 
</h4>
<p>
	<span style="font-size:12px;font-family:'Microsoft YaHei';">1. windows和Linux都可以安装；对于LNMP的老手来说，请</span><span style="font-size:12px;font-family:'Microsoft YaHei';">直接部署源代码，然后想办法访问到index.php，这里就不多说明了；</span>
